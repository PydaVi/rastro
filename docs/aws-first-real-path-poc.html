O caminho de comprometimento

O agente partiu de um IAM User com permissões mínimas e chegou ao acesso de um objeto S3 sensível em 3 passos, passando por assunção de role e validação de permissão efetiva antes do acesso final.

attack path graph — rastro v0.1 objective_met: true

// passo a passo

01
sts:GetCallerIdentity · iam:ListRoles
O agente resolve a identidade efetiva em uso e enumera as roles disponíveis na conta. Nenhuma suposição sobre o ambiente — tudo descoberto dinamicamente a partir das credenciais fornecidas.
→ identidade resolvida · roles enumeradas com sucesso
02
sts:AssumeRole · iam:SimulatePrincipalPolicy
O agente assume a role alvo identificada no step anterior. Antes de tentar o acesso final, valida a permissão efetiva via SimulatePrincipalPolicy — comprovando que s3:GetObject é allowed sem executar a ação ainda.
→ role assumida · decisão: allowed · credenciais temporárias emitidas
03
s3:GetObject
Com credenciais temporárias da role assumida, o agente acessa o objeto S3 alvo. O conteúdo foi capturado localmente e redigido nos artefatos sanitizados. Objetivo atingido.
→ objeto acessado · preview capturado · objective_met: true

Limites atuais e próximos passos

Esta PoC representa o primeiro corte de execução real. O que ainda não foi validado nesta etapa:

// limites desta poc

  • Seleção dinâmica de role a partir das roles descobertas
  • Descoberta dinâmica do recurso final sem conhecimento prévio
  • Competição entre múltiplos caminhos possíveis
  • Decisão autônoma entre caminhos concorrentes
  • Cobertura mais ampla de superfícies AWS

// fase 3 — próximos paths

  • PassRole chain via Lambda — T1548
  • EC2 instance profile → terraform state → secrets
  • Cross-account trust policy abuse
  • Planner escolhendo entre caminhos alternativos
  • Relatório apresentável para cliente real
// barra de maturidade

A fase AWS estará madura quando: 3+ paths end-to-end reais auditados em conta de lab, planner demonstrando escolha entre caminhos alternativos, e relatório gerado apresentável para engenheiro sênior sem explicação adicional.
